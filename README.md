# RentPath CF-Sentry

A [Sentry](https://sentry.io/) Client for Cloudflare Workers

## Installation

**Yarn**

```bash
yarn add rentpath-cf-sentry
```

**NPM**

```bash
npm install rentpath-cf-sentry --save
```
**Deno**
```typescript
import { log } from 'https://denopkg.com/rentpath/rentpath-cf-sentry/index.ts';
```
